# Raspberry Pi Smart Clock

A React-based smart clock application built for Raspberry Pi designed for continuous always-on display.

![Smart Clock](https://github.com/user-attachments/assets/6c34225a-b09b-4390-9ded-06280c461435)

## Features
- Real-time clock display
- Weather information
- Spotify playback status
- Google Calendar integration

## Technologies
- React
- JavaScript
- Raspberry Pi
- REST APIs

## Goal
Build a lightweight smart display optimized for low-power hardware.
